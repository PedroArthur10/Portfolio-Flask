Projetos

Dashboard de Chamados

Este projeto consiste em um dashboard interativo para monitorar e gerenciar chamados em uma empresa. O dashboard foi desenvolvido usando Python e bibliotecas como Dash e Plotly, permitindo a visualização de métricas importantes, como o tempo médio de resolução de chamados, número de chamados abertos, fechados, pendentes, e a distribuição de chamados por setor. Com filtros interativos e gráficos dinâmicos, o dashboard ajuda os gerentes a tomar decisões informadas e melhorar a eficiência do atendimento ao cliente.

Dashboard de Pagamentos de Notas Fiscais

Este projeto é um dashboard desenvolvido para acompanhar e gerenciar pagamentos de notas fiscais em uma empresa. Utilizando ferramentas como Power BI e SQL, o dashboard fornece uma visão abrangente sobre o status dos pagamentos, incluindo notas fiscais pendentes, pagas e em atraso. Ele também inclui recursos de drill-down para analisar informações detalhadas por fornecedor, data de vencimento, e departamento responsável, facilitando o controle financeiro e garantindo a conformidade com os prazos de pagamento.
